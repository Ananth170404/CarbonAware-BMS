# CarbonAware-BMS

**AI-Powered Predictive & Carbon-Aware Building Management System**

CarbonAware-BMS transforms traditional commercial buildings into intelligent, grid-aware, sustainability-driven assets.
It integrates real-time IoT monitoring, anomaly detection, AI forecasting, and carbon-aware load optimization into a scalable cloud platform.

---

##  Problem

Modern enterprise buildings face:

*  Hidden energy inefficiencies
*  High electricity costs (top 3 operational expense)
*  Reactive “fail-and-fix” maintenance
*  Increasing ESG and carbon reporting pressure
*  Undetected equipment degradation

Energy waste often remains invisible until it becomes expensive.

---

## Solution

CarbonAware-BMS provides:

*  **Real-Time Grid-Level Monitoring**
*  **AI-Based Anomaly Detection**
*  **24-Hour Energy Forecasting (95% Confidence Interval)**
*  **Live Carbon Emission Tracking (kWh → kgCO₂e)**
*  **Carbon-Aware Load Shifting ("Green Window")**
*  **Automated ESG-Ready Data Pipeline**

We shift buildings from:

> Reactive facilities → Predictive, sustainable assets.

---

##  Architecture Overview

### Data Flow

IoT Sensors
→ Azure IoT Hub
→ Azure Functions (real-time processing)
→ Azure SQL Database (hot path)
→ Azure Data Lake (cold path)
→ Azure Machine Learning (model retraining)
→ Power BI Dashboard
